    Consider this graph

    Now it is obvious that there is a line segment of coordinates and also angle and length. Normally we just draw lines by specifying coordinates but if we are given angles and length of line, How can we draw the line?

    The answer is simple and easy to understand.

    We assume and convert the line into a right angled triangle. Every line can be converted into a right angled triangle.In the given figure z is the hypotenuse and also the length of line, Y is altitude and X is base.By using trignometeric ratios on 'Fita' angle,

  • Sin angle  = Y/z
  • Cos angle = X/z
  • Tan angle = Y/X

    These are simple and crude ratios but for using them in programming we have to refine them for utilization. In VB Sin, Cos and Tan functions gives output in radians, for degrees the equation will become,

  • Sin(3.1416*angle/180) = Y/z
  • Cos(3.1416*angle/180) = X/z

    For finding X and Y

  • Y = Sin(3.1416*angle/180) *z
  • X = Cos(3.1416*angle/180)*z

    These two ratios are sufficient for finding coordinates or angles. First one can be used to find top value and second one can be used to find left value. But it is not fully done. Consider the diagram batween graph and screen

    It is evident from this figure that in screen the value of Y coordinate has inverse effect on graph, to solve this problem the equation for finding Y changes.

  • Y = -(Sin(3.1416*angle/180) *z)

    So as the value of Y increases in graph it will decrease in screen

    But we are not finished yet. We converted line into a triangle but the location of output line is a problem

     It is necessary to ammend above the equation now
  • Y = -(Sin(3.1416 * angle / 180) * z) + Y position
  • X =   Cos(3.1416 * angle / 180)* z + X position

    This is the final Equation and any thing can be extracted from it

 Given below are some problems,

    Q.1 There is a button on a form with top as 500 and left as 600 and there is another button located at 60o from it having 50 pixels distance from each other. Find the left and top of the second button?

    Solution:-

    Y = -(Sin(3.1416*angle/180) *z) + Y position

    X = Cos(3.1416*angle/180)*z + X position

    top = -(sin(3.1416 * 60 / 180) * 50) +500

    left = cos(3.1416 * 60 / 180) * 50 + 600

    top = 456

    left = 624

    Q.2 We have a form and we want to draw an oval on it having centre at 200 left and 256 top and also width of 100 and height of 50. Write the program if height of form is 480 and width is 640?

    Solution:-

    Me.AutoRedraw = True
    Me.ScaleMode = 3
    Me.ScaleWidth = 640
    Me.ScaleHeight = 480
    Dim k As Integer
    Dim X As Integer, Y As Integer
    For k = 0 To 360
        Y = -(Sin(3.1416 * k / 180) * 50) + 256
        X = Cos(3.1416 * k / 180) * 100 + 200
        Me.PSet (X, Y)
